## We are One

### Requirements 
- Node 15.3.0
- Postgres

### Prerequisite

Install the `ts-node` `typeorm` and `nodemon` globally by running the below commands.
- `npm install -g ts-node`
- `npm install -g typeorm`
- `npm install -g nodemon`

### Available commands

1. `yarn start`
2. `yarn build`
3. `yarn start:prod`
4. `yarn typeorm`
5. `yarn typeorm schema:sync`
